## Chapter 1 Getting Started
"The way to learn a new programming language is to write programs."
### 1.1 Writing a Simple C++ Program

```c++
#include <iostream>
int main()
{
    return 0;
}
```
编译：
```shell
cc main.cpp # 输出 a.out 二进制可执行文件
# 或者 g++
g++ -o main main.cpp
# 输出 main 二进制可执行文件
```
